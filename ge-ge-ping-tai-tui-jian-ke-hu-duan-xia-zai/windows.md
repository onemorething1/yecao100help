## 退出客户端

{% hint style="danger" %}
如果你发现，不开客户端的情况下无法正常上网，则说明你没有正常退出客户端
{% endhint %}

{% hint style="danger" %}
请确保每次退出客户端的时候，都是右键点击图标，然后选择”退出“菜单
{% endhint %}

{% hint style="danger" %}
如果不开客户端的时候无法上网，请再打开一下客户端，然后重新退出
{% endhint %}
